# AIJournalCreator プロジェクトルール

## スプレッドシート操作とGAS開発の方針

本プロジェクトにおいてAIがツールを構築するにあたって、Google スプレッドシートの操作および Google Apps Script (GAS) の開発は、以下の基本方針に従って行います。

1. **スプレッドシートの操作はGAS経由で行う**
   - スプレッドシート上のシートの作成や読み込みは、google workspace cli （gws）経由で実行します。

2. **GASの開発はローカル環境の `gas_proj` フォルダで行う**
   - GASのコーディングはブラウザ上のエディタではなく、ローカルの `gas_proj` フォルダ内で実施します。
   - 作成・編集したコードは、Google公式ツールである `clasp` (`clasp push` / `clasp pull` 等) を使用してスプレッドシートのプロジェクトに反映させます。

3. **不要なファイル・権限設定を行わない**
   - API認証用のJSONキーファイルや、Node.jsで直接操作を行うためのパッケージ（`package.json`, `node_modules`等）は本プロジェクトのルートディレクトリに配置・使用しません。
